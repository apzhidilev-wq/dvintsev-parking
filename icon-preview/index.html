Logo Preview · iOS Home Screen

Пять вариантов логотипа для «Добавить на экран Домой». Все — 88×88 в режиме предпросмотра, фактический файл будет 180×180 для apple-touch-icon (iOS) и 512×512 для Android. Каждый показан на iOS-обоях с подписью «Паркинг». iOS автоматически делает squircle-маску.

1 · Brand Direct
текущий логотип, как есть

Точное повторение логотипа из шапки приложения — синий square, белая «P», точка-акцент. Полная преемственность брендинга.

Паркинг
→ Pick #1 — максимум consistency
2 · Premium Gradient
градиент acc → acc-deep

Та же геометрия, но фон — диагональный градиент iOS Blue → Apollo Deep. На светлых обоях iOS даёт depth и premium-feel. Этот стиль использует Apple для system apps.

Паркинг
→ Pick #2 — добавляет премиум-нюанс, остаётся узнаваемо
3 · Parking Sign
P в кружочке — иконично

Стилизация под дорожный знак «Парковка». Белый круг с синей P. Вне контекста сразу читается «парковка», даже если не знаешь приложения. Для public-facing более universal.

Паркинг
→ Pick #3 — universal, читается «парковка» сразу
4 · Layered Architecture
полки + P

Stylized 3-уровневая «многоуровневая парковка» (БЦ Двинцев) + P over. Передаёт смысл «учёт парковки», не только «парковка вообще». Чуть больше деталей.

Паркинг
→ Pick #4 — намёк на многоуровневую парковку
5 · Mono Black
премиум-минимализм

Чёрный фон, белая P, синяя точка-акцент. Тёмный «luxury» вид — как у Stripe или Linear. Сильно отличается от остальных синих app-иконок на главном экране → легко найти глазами.

Паркинг
→ Pick #5 — выделяется на фоне других иконок
