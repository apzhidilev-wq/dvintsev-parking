# БЦ «Двинцев» — Учёт парковки

Single-file веб-приложение для учёта парковочных мест бизнес-центра.

## Возможности

- Интерактивная схема паркинга на PDF (PDF.js)
- Учёт ячеек по секциям (А / В / С / Наземная / Вне ячеек)
- Поиск по компании / ФИО / гос. номеру (с нормализацией кириллица↔латиница)
- Ролевая модель (admin / operator / viewer)
- Аудит-лог изменений
- Экспорт/импорт данных в JSON
- QR-таблички с deep-link на ячейку
- Pinch-zoom + touch-pan на мобильных
- Telegram-уведомления о событиях
- 2 темы: тёмная и «Тёплая бумага»

## Безопасность

- PBKDF2-SHA256 100k итераций для паролей (с per-user salt)
- Подписанные сессии (привязка к user-agent + хешу пароля)
- При первом запуске генерируются случайные пароли — показываются один раз

## Запуск локально

```bash
python3 -m http.server 8000
# открыть http://localhost:8000/parking.html
```

## Стек

Чистый HTML/CSS/JS, без сборки. PDF.js через CDN. Все данные — в localStorage.

## Файлы

- `parking.html` — основное приложение
- `scheme.pdf` — план подземного паркинга
- `scheme_ground.pdf` — план наземного паркинга
